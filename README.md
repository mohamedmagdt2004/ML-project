# Machine Learning Project

This project is about applying machine learning techniques on the California Housing dataset.

The project includes:
- Data preprocessing
- Data analysis
- Linear Regression
- Logistic Regression
- K-Means Clustering

Libraries used:
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

The models were evaluated using different metrics such as:
- RMSE
- R2 Score
- Accuracy
- Silhouette Score
